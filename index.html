Bridge Fatigue Monitoring Using Hybrid Digital Twin

Masters – Individual Project

📄 Full Paper accepted at IABMAS 2026 Conference

Problem Identified: Utilizing existing camera infrastructure for real-time structural health monitoring.

Built a real-time structural health monitoring system applicable to any bridge with existing camera infrastructure. The hybrid approach combines a modified Lighthill-Whitham-Richards (LWR) traffic flow model with Random Forest prediction (R² = 0.88). The system captures live video streaming, processes it using YOLO computer vision for vehicle detection, utilizes Open-Meteo weather API for real-time weather integration, and runs Monte Carlo simulations to quantify uncertainty in fatigue forecasting. Demonstrated through an interactive Streamlit dashboard with maintenance scheduling recommendations.

Solution Proposed Framework for shifting from reactive bridge inspections to proactive, data-driven maintenance planning
Key Results R² = 0.88 prediction accuracy, real-time fatigue scoring with uncertainty bounds
Tech Stack Python, OpenCV, YOLO, Scikit-learn, Selenium, Streamlit, Weather API
Domain Infrastructure Health, Predictive Maintenance

AI-Assisted Data Center Routing for Urban Heat Island Mitigation

Masters – Individual Project

📄 Full Paper accepted at ANNSIM 2026 Conference

Problem Identified: An energy-efficient data center prompt routing system that reduces urban heat island effects.

Developed an intelligent workload routing system that balances energy efficiency, latency, and localized heat impact across geographically distributed data centers. Trained MLR and ANN models on 736,000+ hourly weather samples from Open-Meteo API, using Bayesian Optimization for neural network hyperparameter tuning. Built physics-based Urban Heat Island (delta-T) model and analytical latency model combining Haversine propagation with M/M/1 queueing theory. The multi-objective routing engine demonstrates improved heat distribution with minimal performance penalty.

Solution Proposed Reduced localized urban heating from data centers while maintaining service quality
Key Results 736k+ samples trained, physics-based UHI modeling, multi-objective optimization

Bayesian Spatio-temporal Taxi Demand Modeling & Geosimulation

Post Masters – Research Project | UCF Urban Digital Twin Lab

Problem Identified: Urban taxi demand has excess zeros, overdispersion, and complex spatial-temporal dependencies that standard models fail to capture.

Built a Bayesian Zero-Inflated Negative Binomial (ZINB) spatio-temporal model using Nearest Neighbor Gaussian Processes (NNGP) for scalable spatial inference on NYC taxi pickup counts by zone and hour. Integrated Pólya Gamma data augmentation for efficient Gibbs sampling with SVD-based linear algebra. Implemented a What-if geosimulation layer for counterfactual scenario analysis (peak-hour demand, weekend policy shifts) with posterior uncertainty quantification. Benchmarked against R INLA with BYM2 spatial effects and RW1 temporal component.

Solution Proposed Modular ZINB-NNGP sampler with interpretable What-if scenario responses for urban digital twin applications
Key Results Improved out-of-sample predictive fit vs. INLA baseline; spatial hotspots in Midtown & airports identified
Tech Stack R, BayesLogit, Matrix, mvtnorm, INLA, R Markdown
Domain Urban Mobility, Bayesian Inference, Geosimulation

Helicopter Fleet Readiness and Workforce Fatigue Analysis

Masters – Group Project

Problem Identified: How maintainer fatigue impacts operational readiness and safety in Army Aviation helicopter fleets.

Investigated the correlation between maintainer fatigue and Army Aviation helicopter readiness using 2,865 maintenance records across 37 aircraft. Initial linear regression showed weak correlation, so we transitioned to Random Forest which captured non-linear relationships. Generated synthetic fatigue survey data for 341 personnel using CTGAN from Synthetic Data Vault. Ran 500 Monte Carlo simulations with Gaussian noise to validate findings and quantify uncertainty in workforce allocation strategies.

Solution Proposed Quantified how maintainer fatigue impacts operational readiness and safety
Key Results Weak correlation between fatigue and fleet availability; validated synthetic data requirements for human factors
Tech Stack Python, Pandas, Scikit-learn, SDV/CTGAN, NumPy, SciPy
Domain Military Operations, Workforce Optimization, Maintenance

Terrain-Aware Battlefield Simulation Using Finite State Machines

Masters – Group Project

Problem Identified: How terrain constraints affect tactical decision-making in military simulations.

Designed a Finite State Machine (FSM) Mathematical Model framework for dynamic military unit management integrating real terrain data. Processed USGS Digital Elevation Model data in QGIS to calculate slope percentages and classify terrain into four categories (Flat, Valley, Hilly, Mountainous). Units automatically transition between Idle and Action states based on computed metrics like Force Output, Velocity, and Power. Monte Carlo sensitivity analysis in @Risk identified slope percentage as the primary risk factor affecting unit performance.

Solution Proposed Automated tactical decision-making based on real terrain constraints
Key Results Slope identified as primary risk driver; automated state transitions reduce manual intervention
Tech Stack QGIS, @Risk, Python, USGS DEM, Excel
Domain Defense Simulation, Decision Support
